Reporting a Bug

A good bug report includes:

  • Exact steps to reproduce the issue
  • What you expected vs what happened
  • Any error messages or screenshots
  • Browser, OS, and app version
Common Issues

Try a hard refresh (Ctrl+Shift+R). Check that the server process is running and no other app is using the same port. Restart the app if needed.

Check that auto-save is working — look for the "Saved" indicator in the topbar. If it shows an error, the database file may be locked or the instance/ folder may lack write permissions.

Settings are saved to localStorage. Make sure you're using the same browser profile and that private/incognito mode is not active (it clears storage on close).

Clear the browser cache (Ctrl+Shift+Del) and reload. If the issue persists, try a different browser to determine if it's browser-specific.

Stop the app, delete instance/app.db, then restart. Settings and configuration files are unaffected unless you delete them separately.
